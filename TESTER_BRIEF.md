# Authentic Steps For Youth — Tester Brief

Welcome, and thank you for helping test **Authentic Steps For Youth** before it goes live.
This document tells you everything you need to install the app, what to try, and how to send
feedback.

---

## What is this app?

Authentic Steps For Youth is a mobile app designed to support young people's mental health and
wellbeing. It provides:

- **Guided rituals** — short daily practices to start or close your day with intention
- **Journaling** — a private space to reflect and track how you're feeling over time
- **Grounding exercises** — in-the-moment tools when things feel overwhelming
- **I Am statements** — affirmations and self-identity prompts

---

## Installing on iPhone (iOS via TestFlight)

1. **Install TestFlight** — download it free from the App Store if you don't already have it.
2. **Accept your invite** — you'll receive an email from Apple/TestFlight with a link to join
   the beta. Tap "View in TestFlight" in the email.
3. **Install the app** — tap "Install" inside TestFlight. The app appears on your home screen
   once downloaded.
4. **Sign up** — open the app and create an account with your email (or use Google/Apple
   sign-in). No invite code is needed; sign-up is open.

> If you haven't received a TestFlight invite, let us know and we'll add your Apple ID email.

---

## Installing on Android (APK direct install)

1. **Download the APK** — tap the link below to download the `.apk` file directly to your device:

   **[Download Authentic Steps For Youth v1.0.0 (APK)](https://authentic-steps-youth.replit.app/api/storage/public-objects/builds/authentic-steps-1.0.0-build4.apk)**

   If that link doesn't work, email **hello@authenticsteps.com.au** and we'll send you the file directly.
2. **Allow installs from unknown sources** — Android blocks this by default for safety:
   - **Android 8+:** when prompted after opening the file, tap **Settings** → enable
     "Allow from this source" for your browser or file manager, then go back and tap Install.
   - **Samsung devices:** Settings → Apps → Special app access → Install unknown apps.
3. **Install the APK** — open the downloaded file and tap Install. You may see a Play Protect
   warning — tap "Install anyway."
4. **Sign up** — open the app and create a free account. No invite code required.

> Note: future updates won't be automatic on Android (unlike TestFlight). We'll share a new
> link when there's a significant update to test.

## What to try

Work through these in order if you can — they cover the core experience:

### First run
- [ ] Create a new account (sign up with email or social)
- [ ] Watch the welcome/intro and complete the onboarding flow
- [ ] Reach the home screen

### Daily ritual
- [ ] Start a ritual from the home screen
- [ ] Watch the intro video, set an intention, and complete the session
- [ ] Return the next day to see if the ritual prompt changes

### Journal
- [ ] Write a journal entry from the Journal tab
- [ ] Add a mood / how-are-you-feeling check-in
- [ ] View a past entry

### Grounding
- [ ] Open the Grounding tool from the Toolbox tab
- [ ] Complete a grounding exercise session

### I Am statements
- [ ] Find the I Am section in the Toolbox
- [ ] Scroll through and tap a few affirmations

### Profile and settings
- [ ] Open the Profile tab
- [ ] Check that your name/avatar shows correctly
- [ ] Toggle a notification setting

### Feedback (most important!)
- [ ] Tap **Profile → Support & Feedback → Report a problem**
- [ ] Type a short test message and submit it — this confirms the feedback pipeline works

## Things we're NOT testing yet

To keep the focus clear, please ignore (or note but don't prioritise):

- Visual polish, font sizes, colour choices — design refinements come after core feedback
- App Store listing, screenshots, ratings prompts

---

## How to submit feedback

**Preferred method — in-app form:**
Profile tab → Support & Feedback → "Report a problem" → type your message → Submit.

Your report goes directly to our team's task tracker. Include:
- What you were doing when the issue happened
- What you expected vs. what actually happened
- Your device model and iOS/Android version (visible in your phone's Settings → About)

**Fallback — if the in-app form doesn't work:**
Email us at **hello@authenticsteps.com.au** with the same details. The app will offer this
automatically if the in-app submission fails.

---

## Known limitations (not bugs)

- Videos require an internet connection — you'll see a clear offline message if there's no
  connectivity (this is by design, videos are streamed to keep the app size small)
- The app is English-only for now
